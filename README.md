# 🛒 Shree Laxmi Bhandar - Fresh Grocery Store

A modern, responsive website for a local grocery store in Mithapur, Bihar, India.

## 📋 Features

✅ **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
✅ **Modern UI/UX** - Beautiful gradient backgrounds and smooth animations
✅ **Product Showcase** - Display your grocery items with prices and descriptions
✅ **WhatsApp Integration** - Direct WhatsApp order links for each product
✅ **Fast Loading** - Pure HTML/CSS, no dependencies or external libraries
✅ **SEO Friendly** - Semantic HTML structure for better search engine visibility
✅ **Navigation** - Sticky navigation bar for easy access to sections
✅ **Contact Information** - Multiple ways to contact the store

## 🎨 Sections

1. **Header** - Store branding with name and tagline
2. **Navigation** - Sticky navbar for easy navigation
3. **Hero Section** - Eye-catching banner with call-to-action
4. **Products** - Grid layout showcasing all grocery items
5. **About** - Information about the store and its benefits
6. **Contact** - Multiple contact methods and store location
7. **Footer** - Copyright information

- **Mobile** (≤480px): Single column layout for optimal mobile experience

## 🚀 Getting Started

### Option 1: Local Setup
1. Clone the repository
2. Open `index.html` in your web browser
3. That's it! The website is ready to use

### Option 2: GitHub Pages
1. Go to repository Settings
2. Navigate to Pages section
3. Select main branch as source
4. Your site will be live at `https://shubhamrkscp-boop.github.io/Shree-laxmi-bhandar/`

## 🔧 Customization
